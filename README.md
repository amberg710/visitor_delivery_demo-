# Visitor & Delivery Hub

A modern visitor management and reception dashboard built with **Flask** and **Google Sheets**.

The system tracks visitor check-ins, scheduled arrivals, badge usage, deliveries, and analytics insights for reception teams.

---

## Features

- Visitor check-in system
- Scheduled visitor management
- Walk-in vs scheduled analytics
- Badge tracking dashboard
- Delivery logging and parcel storage
- Reception activity analytics
- Real-time dashboard metrics
- Google Sheets backend
- Modern analytics charts

---

## Dashboard Modules

### Visitor Check-In
Reception can register visitors and assign available badges.

### Expected Visitors
Displays scheduled arrivals for the current day with quick check-in.

### Badge Dashboard
Shows badge availability and alerts for badges not returned after hours.

### Deliveries
Logs packages received for employees and tracks storage locations.

### Analytics
Business-style dashboard showing:

- Visitor trends
- Scheduled vs walk-in visitors
- Reception activity
- Purpose breakdown
- Badge utilization
